Маркировка
за 3 секунды

Лэйблер выбирает пробирки нужно вида и маркирует их уникальными штрихкодами. Для контейнеров других форм Лэйблер распечатывает отдельные штрихкоды.

Любая информация
на штрихкоде

Печатайте всю необходимую информацию на штрихкоде: дату и время образца, цифровой код, ФИО пациента, дату рождения пациента и другое

Интеграция с любой
МИС / ЛИС

Лэйблер получает из МИС/ЛИС информацию о назначенных исследованиях для конкретного пациента.
